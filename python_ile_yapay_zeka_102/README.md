Projeyi yerel makinenize klonlamak ve bağımlılıkları yüklemek için aşağıdaki adımları izleyin:

### 1. Depoyu Klonlayın
```sh 
git clone https://github.com/cavitbatusoylu/python_techcareer_2026_201_1.git
```

### 2. Bağımlılıkları Oluşturun
Gerekli kütüphaneleri otomatik olarak oluşturmak için shell scriptlerini kullanabilirsiniz:

**Otomatik Gereksinim Oluşturucu:**

